## 功能描述
获取各个平台热搜信息

## 请求地址
``` bash
GET /api/openapi/v1/hotTrend/{platform}
```

## 公共头参数
| 参数名 | 类型 | 是否必须 | 描述 |
| :----- | :--- | :------- | :--- |
| `app_id` | string | 是 | 应用ID |
| `Authorization` | string | 是 | 访问令牌，格式为 `Bearer <access_token>` |
| `Content-Type` | string | 是 | 请求体的内容类型，通常为 `application/json` |

## 请求参数
| 参数名 | 类型 | 是否必须 | 描述 |
| :----- | :--- | :------- | :--- |
| `platform` | string | 是 | 平台名称，如 `weibo`、`twitter`、`instagram` 等 |
