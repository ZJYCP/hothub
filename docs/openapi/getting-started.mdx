 
# 入门指南

你好，开发者！欢迎使用我们的开放平台API。通过这些API，你可以便捷地集成我们的服务，构建功能强大的应用。

本指南将帮助你快速了解如何接入并开始调用我们的API。

## 准备工作

在开始调用API之前，你需要完成以下准备工作：

1.  **注册成为开发者**: 如果你还没有注册，请先访问完成注册。
2.  **创建应用并获取凭证**: 注册成功后，请创建一个新的应用。创建应用后，你将获得一对重要的凭证：
    - `app_id` (应用ID)
    - `app_secret` (应用密钥)

:::caution
`app_secret` 是你应用的绝密信息，请务必妥善保管，不要在客户端代码中直接暴露，也不要泄露给任何第三方。
:::

## 接入流程概述

使用我们的API通常遵循以下核心流程：

1.  **获取访问令牌 (Access Token)**: 使用你的 `app_id` 和 `app_secret` 调用特定的认证接口 `/api/openapi/auth/token` 来换取一个有时效性的 `access_token`。
2.  **调用业务接口**: 在后续调用其他业务接口时，需要在请求的 HTTP Header 中携带上一步获取到的 `access_token` 进行身份验证。
3.  **处理令牌过期**: `access_token` 有有效期。当令牌过期时，你需要重新执行第一步来获取新的令牌。

这个流程确保了你的应用身份被验证，并且业务接口的调用是安全的。
